# Memories

## Installing on Local Machine

### Prerequisites

- [Node.js](https://nodejs.org/en/)

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Heritsam/memories.git
   ```

2. Go to the project directory
   ```sh
   cd memories
   ```

3. Install NPM packages
   ```sh
   npm install
   ```

## Usage

1. Run the app
   ```sh
   npm run dev
   ```

2. Open the app on your browser
   ```sh
   http://localhost:5173/
   ```
